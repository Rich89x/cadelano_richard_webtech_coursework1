The Morse code

The Morse code was invented by the American Samuel Morse (1791 - 1872).
During the 1830, after the invention of the first telegraph in England,
that was able to send messages using electric signals to line up magnetic
needles on a grid containing the letters of the alphabet, Samuel Morse and
Alfred Vail developed a single-circuit telegraph that sent the signal across
the wire by pushing the operator key down.

To transmit messages through their telegraph Morse and Veil developed the
Morse Code: this code assigns letters and numbers to dots and dashes, dots
for short impulses and dashes for long impulses, this code is based on the
frequency of use: letters used often = simple code.
The most common message in morse code is the S.O.S. (...---...).
